CSS Loader Tutorial

In this tutorial, you'll learn how to create a stylish CSS loader using pure CSS. This loader is perfect for indicating loading states in your web applications.

Live Demo

Code Explanation

The loader is created using a combination of CSS variables, gradients, and animations. Here's a breakdown of the code:

  1. CSS Variables: The loader uses CSS variables (--R, --g1, and --g2) to define its size and colors.
  2. Gradients: The loader uses multiple radial-gradient backgrounds to create its unique design.
  3. Animation: The loader spins continuously using a CSS animation (@keyframes).

Full Code

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>CSS Loader</title>
  <style>
    body {
      display: flex;
      justify-content: center;
      align-items: center;
      height: 100vh;
      margin: 0;
      background: #1e1e1e;
    }
    .loader {
      --R: 60px;
      --g1: #514b82 96%, #0000;
      --g2: #eeeeee 96%, #0000;
      width: calc(2 * var(--R));
      aspect-ratio: 1;
      border-radius: 50%;
      display: grid;
      -webkit-mask: linear-gradient(#000 0 0);
      animation: l30 2s infinite linear;
    }
    /* ... rest of the CSS ... */
  </style>
</head>
<body>
  <div class="loader"></div>
</body>
</html>
